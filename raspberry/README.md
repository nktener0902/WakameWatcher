## How to use

### Prerequisite

* Java (11>=)
* Gradle (5>=)

### For mac

Before running this application, a user needs to install `imagesnap` that is an image-capture application.

```bash
brew install imagesnap
tccutil reset Camera
```

After that, the user can run this application.

```bash
gradle bootRun
```
